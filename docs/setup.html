Setting up RCHpr

You will need Git, node.js, electron and some kind of editor supporting .json files (notepad will do lol). I will link those later rn I cba lol

  1. If you're on Mac/Linux, open Terminal. On Windows, go to the Start Menu, type in cmd.exe and press Enter.
  2. Run git clone https://github.com/ICTman1076/RCHpr.git to download the repo
  3. Run cd RCHpr to go to the folder
  4. Run (Windows only) copy config.json.example config.json or (on macOS/Linux) cp config.json.example config.json to initialise the configuration
  5. Run npm install to install all the requirements
  6. Open a file browser - this could be Finder on Mac or Windows Explorer
  7. Now, go to the folder where RCHpr was downloaded
  8. Open config.json with your chosen .json editor
  9. Many settings in config.json are very broken due to the new preset feature. To add presets, follow the examples given in the games array - name is what will be displayed, code (which must be all lower case) should be unique, and will also display the images you set up in the discord rich presence developer section, where the name is the same as code.
  10. IMPORTANT: At the bottom there is a "clientID". This is currently set to a test id owned by me and so is unreliable. If you want your own, create a rich presence app as shown here and use the client ID generated from that. If you do create your own you can upload your own images, instead of suffering with mine lol
  11. Save config.json and close Atom
  12. Make sure Discord is running (only one - you can't have more than one open!)
  13. Go back to Cmd or Terminal, and run npm start
  14. You have successfully set up and started RCHpr!
